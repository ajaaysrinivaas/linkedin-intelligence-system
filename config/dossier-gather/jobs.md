# Job Searches to Track

Configure job search queries for monitoring healthcare AI, sports tech, and data roles.

## Search Queries

```yaml
jobs:
  - title: "Healthcare AI Engineer"
    keywords: "healthcare AI engineer"
    location: "Remote"
    
  - title: "Sports Data Analyst"
    keywords: "sports data analyst"
    location: "Remote"
    
  - title: "Physiotherapy Tech Roles"
    keywords: "physiotherapy software developer"
    location: "Remote"
    
  - title: "Clinical AI Researcher"
    keywords: "clinical AI researcher machine learning"
    location: "Remote"
```

## What Gets Tracked

For each search query:
- New job postings
- Salary ranges
- Required skills
- Company hiring trends
- Job description changes

## Output

Results saved to: `data/dossiers/active/jobs/{timestamp}.json`
