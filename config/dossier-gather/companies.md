# Companies to Watch

Configure healthcare AI and sports tech companies to track for intelligence gathering.

## Watch List

```yaml
companies:
  - name: "Acme Healthcare"
    search_terms: "Acme Healthcare AI"
    
  - name: "HealthTech Startup B"
    search_terms: "HealthTech Startup B funding"
    
  - name: "MedAI Platform"
    search_terms: "MedAI Platform"
    
  - name: "Clyniq"
    search_terms: "Clyniq healthcare digitization"
```

## What Gets Tracked

For each company:
- Recent company posts
- Job openings
- Company profile updates
- Funding announcements
- Team changes

## Output

Results saved to: `data/dossiers/active/companies/{timestamp}.json`
